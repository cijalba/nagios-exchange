# nagios-exchange
Nagios exchange scripts for monitoring

Published in nagios exchange between 2016-2019.
All this scripts were checked for compatibility in a great range of Unix/Linux OS and on cygwin for it's use in Windows *NIX emulation environments too.

check_klms - Kaspersky for Linux Mail Security (KLMS) plugin [bash shell]
check_mount - Linux/Unix generic plugin to check for FS mounts (detects double mount issues, quite flexible) [POSIX shell]
check_lpar2rrd - AIX LPAR2RRD plugin to check for CPU status of an IBM PowerVM LPAR or PowerVM POOL of servers [korn shell]
